# 树

## 代码定义

基本二叉树：
``` C++
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
};
```

多叉树：
``` C++
struct struct Node {
    int val;
    vector<Node*> children;
};
```

## DFS用递归，也可以手动维护栈，防止爆栈

DFS针对树的处理，永远只有：1. 处理当前节点，2. 处理左子树，3. 处理右子树，只是顺序有别。
前中后序只是取决于步骤1放在哪，放在最前面就是前序，放在最后面就是后序

具体哪个序的DFS，取决于处理当前节点的信息是否依赖子树结果，如果要，则先递归再处理，采用后序，否则采用前序


## BFS是队列逐层访问，一般队列 + while循环一个个访问
``` C++
void bfs(TreeNode* root) {
    queue<TreeNode*> q;
    if (root) q.push(root);

    while (!q.empty()) {
        auto* node = q.front(); q.pop();
        process(node);

        if (node->left)  q.push(node->left);
        if (node->right) q.push(node->right);
    }
}
```
