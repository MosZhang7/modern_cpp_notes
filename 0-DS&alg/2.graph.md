# 图

## 图的存储

邻接表是图存储的**事实标准**，如果图确定非常小，可能可以用邻接矩阵
``` C++
// 邻接表：
vector<vector<int>> graph(n);
graph[u].push_back(v);

// 加权图
vector<vector<pair<int,int>>> graph(n);
graph[u].push_back({v, weight});
```

## 图的遍历

只要是图，就必须标记访问过，因为可能绕圈

### 图dfs(u)：标记所有从u出发能到达的点
``` C++
void dfs(int u, vector<vector<int>>& graph, vector<bool>& visited) {
    visited[u] = true;
    process(u);

    for (int v : graph[u]) {
        if (!visited[v]) {
            dfs(v, graph, visited);
        }
    }
}
```

### 图BFS：无权最短路
``` C++
vector<int> bfs(int n, vector<vector<int>> &graph, int start) {
  vector<int> dist(n, -1);
  queue<int> q;

  dist[start] = 0;
  q.push(start);

  while (!q.empty()) {
    int u = q.front();
    q.pop();
    for (int v : graph[u]) {
      if (dist[v] == -1) {
        dist[v] = dist[u] + 1;
        q.push(v);
      }
    }
  }
}
```

### Dijkstra：加权最短路
``` C++
  vector<int> dist(n, INF);
  priority_queue<pair<int, int>, vector<pair<int, int>>, greater<>> pq;

  dist[start] = 0;
  pq.push({0, start});

  while (!pq.empty()) {
    auto [d, u] = pq.top();
    pq.pop();
    if (d > dist[u])
      continue;

    for (auto [v, w] : graph[u]) {
      if (dist[v] > dist[u] + w) {
        dist[v] = dist[u] + w;
        pq.push({dist[v], v});
      }
    }
  }
```

## 拓扑排序

处理依赖，启动顺序问题。一般先准备好入度依赖图DAG，把没有任何依赖的先处理，并移除DAG，直到最后所有的无依赖item被处理完后，再看图是否为空
``` C++
  vector<int> topoSort(int n, vector<vector<int>> &graph) {
    vector<int> indeg(n);
    queue<int> q;

    for (u->v)
      indeg[v]++;

    for (int i = 0; i < n; ++i)
      if (indeg[i] == 0)
        q.push(i);

    while (!q.empty()) {
      int u = q.front();
      q.pop();
      order.push_back(u);

      for (int v : graph[u]) {
        if (--indeg[v] == 0)
          q.push(v);
      }
    }
  }
```

## 并查集：判断连通性
``` C++
struct DSU {
    vector<int> parent;

    DSU(int n): parent(n) {
        iota(parent.begin(), parent.end(), 0);
    }

    int find(int x) {
        if (parent[x] != x)
            parent[x] = find(parent[x]);
        return parent[x];
    }

    void unite(int x, int y) {
        parent[find(x)] = find(y);
    }

    bool connected(int x, int y) {
        return find(x) == find(y);
    }
};
```
