# 类型推导

## auto：使用模板参数推导规则，推导出干净的值类型，不含const不含引用

auto只给出最原始的值类型，去掉引用去掉cv，数组/函数变为指针，不管类别等，需要自己加
例如：auto&和const auto &&


## decltype：推导出表达式的类型，会包括const和引用类型等

表达式推导规则：

| 表达式值类别  | 返回类型      |
| ------- | --------- |
| lvalue  | `T&`      |
| xvalue  | `T&&`     |
| prvalue | `T` （对象值） |

当仅仅传入变量名而非表达式时，decltype直接返回变量声明的类型(不是auto的纯值类型)以方便。

而加括号之后会成为表达式，会使用表达式推导方式，因此需要小心使用括号，例如
``` C++
int x = 0;

decltype(auto) c1 = (x); // int&   走表达式推导方式
decltype(auto) c2 = x;   // int    走变量推导方式
decltype(auto) c3 = std::move(x) // int&&

// decltype(auto)：走的是decltype的推导方式，auto仅仅做占位符，将表达式替换过来而已
```
