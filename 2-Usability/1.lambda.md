# lambda

C++14起，auto可以用于lambda表达式中的函数传参，C++20起推广到了一般函数。
因为lambda函数是类，是一个重载了operator()的模板类，而C++20开始，为一般函数也自动生成了对应的模板类
``` C++
auto add14 = [](auto x, auto y) -> int {
    return x+y;
}

int add20(auto x, auto y) {
    return x+y;
}

auto i = 5; // type int
auto j = 6; // type int
std::cout << add14(i, j) << std::endl;
std::cout << add20(i, j) << std::endl;
```

lambda本质是一个函数闭包对象。闭包是指有无捕获列表，未捕获时，类内部不保存状态，编译器直接生成一个static普通函数，因此可以直接转换为函数指针进行传递。如果有捕获则不行，因为lambda类内部要保存外界状态，不能转换为函数指针。

注意lambda在捕获时，内部状态是在lambda创建出来时就保存进去了，内部都是const类型，除非加上mutable才会改自己内部的。
